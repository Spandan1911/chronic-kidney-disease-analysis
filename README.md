# chronic-kidney-disease-analysis
Exploratory Data Analysis of Chronic Kidney Disease (CKD) dataset using Python, Pandas, Matplotlib, Seaborn, and Plotly to identify key health indicators, correlations, and disease patterns.

# Dataset Overview

Dataset: Chronic Kidney Disease Dataset

Records: 400 Patients

Objective:
Analyze patient health indicators and identify patterns associated with Chronic Kidney Disease.

# Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Jupyter Notebook
  
# Key Analysis Performed

- Data Cleaning
- Missing Value Handling
- Feature Engineering
- Univariate Analysis
- Bivariate Analysis
- Multivariate Analysis
- Correlation Analysis
- Interactive Visualizations

# Key Insights

• Serum Creatinine is strongly associated with CKD.
• Blood Urea levels are significantly higher in CKD patients.
• Diabetes and Hypertension show strong association with kidney disease.
• Age and Blood Pressure contribute to disease severity.
• Several clinical features exhibit strong correlations useful for predictive modeling.
